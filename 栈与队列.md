# 栈
## 定义
栈是一种仅可在栈顶进行插入和删除操作的数据结构

## 分类
普通栈：线性存储
链栈：链式存储（链表）

## 数组模拟

### 相对STL的优势
1. 快
2. 可以有自己想要的功能[[STL#栈]]

### 模拟
```c++
using namespace std;

int sta[100];//栈
int point;//栈顶指针

int main(){
	int m;//栈的长度
	cin>>m;
	while(m--){
		string com;
		int x;
		cin>>com;
		if(com == ''push''){
			cin>>x;
			sta[++point] = x;//插入
		}
		else if(com == ''pop''){
			point--;
		}
		else if(com == ''empty''){
			if(point == 0){
				printf("YES");//为空
			}
			else {
				printf("NO");//不为空
			}
		}
		else{
			printf("%d",sta[point]);//输出栈顶元素
		} 
	}
	return 0;
}
```
## 队列模拟栈
### 题目
[225. 用队列实现栈 - 力扣（LeetCode）](https://leetcode.cn/problems/implement-stack-using-queues/)

### 两个队列模拟
**思路类似用两个栈模拟队列，不过要注意在给que2赋值时保留que1的最后一个元素（即所需元素）**
```c++
	queue<int> que1;
    queue<int> que2; // 辅助队列，用来备份
    /** Initialize your data structure here. */
    MyStack() {

    }

    /** Push element x onto stack. */
    void push(int x) {
        que1.push(x);
    }

    /** Removes the element on top of the stack and returns that element. */
    int pop() {
        int size = que1.size();
        size--;
        while (size--) { // 将que1 导入que2，但要留下最后一个元素
            que2.push(que1.front());
            que1.pop();
        }

        int result = que1.front(); // 留下的最后一个元素就是要返回的值
        que1.pop();
        que1 = que2;            // 再将que2赋值给que1
        while (!que2.empty()) { // 清空que2
            que2.pop();
        }
        return result;
    }

    /** Get the top element. */
    int top() {
        return que1.back();
    }

    /** Returns whether the stack is empty. */
    bool empty() {
        return que1.empty();
    }
};
```


### 一个队列模拟
**一个队列在模拟栈弹出元素的时候只要将队列头部的元素（除了最后一个元素外） 重新添加到队列尾部，此时再去弹出元素就是栈的顺序了。**
```c++
class MyStack {
public:
    queue<int> que;
    /** Initialize your data structure here. */
    MyStack() {

    }
    /** Push element x onto stack. */
    void push(int x) {
        que.push(x);
    }
    /** Removes the element on top of the stack and returns that element. */
    int pop() {
        int size = que.size();
        size--;
        while (size--) { // 将队列头部的元素（除了最后一个元素外） 重新添加到队列尾部
            que.push(que.front());
            que.pop();
        }
        int result = que.front(); // 此时弹出的元素顺序就是栈的顺序了
        que.pop();
        return result;
    }

    /** Get the top element. */
    int top() {
        return que.back();
    }

    /** Returns whether the stack is empty. */
    bool empty() {
        return que.empty();
    }
};
```
# 队列
## 定义
队列是只允许在队头进行删除，在队尾进行插入的数据结构

## 分类
链式：；链式存储（链表）
循环队列：队尾与队头连在一起
双端队列：队尾和队头都具有插入和删除操作的队列

## 数组模拟

### 相对STL的优势
1. 快
2. 可以添加自己的功能
[[STL#队列]]
### 模拟
```c++
using namespace std;

int que[100];
int headpoint;//头指针
int tailpoint;//尾指针

int main(){
	int m;
	cin>>m;
	while(m--){
		string cmd;
		int x;
		cin>>cmd;
		if(cmd == ''push''){
			que[tailpoint++] = x;
			
		}
		else if(cmd == ''pop''){
			headpoint++;
		}
		else if(cmd == ''empty''){
			if(headpoint < tailpoint){
				printf("NO");
			}
			else{
				printf("YES");
			}
		}
		else{
			printf("%d",que[tailpoint]);
		}
	}
	return 0;
}
```




## 栈模拟队列
### 题目
[232. 用栈实现队列 - 力扣（LeetCode）](https://leetcode.cn/problems/implement-queue-using-stacks/)

### 两个栈来模拟队列
**栈是后入先出，队列先入先出，所以两个栈中一个栈用来存储输入的元素，另一个栈负责改变元素顺序即可**
**同时有必要关注到pop和peek中思路相似的部分，不合适直接粘贴代码，应当考虑函数或类的复用（工业开发）**
```c++
class MyQueue {
public:
    stack<int> stIn;
    stack<int> stOut;
    /** Initialize your data structure here. */
    MyQueue() {

    }
    /** Push element x to the back of queue. */
    void push(int x) {
        stIn.push(x);
    }

    /** Removes the element from in front of queue and returns that element. */
    int pop() {
        // 只有当stOut为空的时候，再从stIn里导入数据（导入stIn全部数据）
        if (stOut.empty()) {
            // 从stIn导入数据直到stIn为空
            while(!stIn.empty()) {
                stOut.push(stIn.top());
                stIn.pop();
            }
        }
        int result = stOut.top();
        stOut.pop();
        return result;
    }

    /** Get the front element. */
    int peek() {
        int res = this->pop(); // 直接使用已有的pop函数
        stOut.push(res); // 因为pop函数弹出了元素res，所以再添加回去
        return res;
    }

    /** Returns whether the queue is empty. */
    bool empty() {
        return stIn.empty() && stOut.empty();
    }
};
```

# 单调栈
## 特性
栈中的元素保证单调性，要注意对元素性质的判断，即哪些元素应当舍去，哪些元素应当压入栈中
## 思考步骤
1. 先尝试暴力枚举
2. 处理枚举结果，看哪些元素用不到
3. 剔除无用元素观察规律
4. 发现单调性

# 单调队列
## 特性
一般在滑动窗口中使用，队首元素为最终值
## 相关题目
[P1886 滑动窗口 /【模板】单调队列 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)](https://www.luogu.com.cn/problem/P1886)
[239. 滑动窗口最大值 - 力扣（LeetCode）](https://leetcode.cn/problems/sliding-window-maximum/)
```cpp
vector<int> maxSlidingWindow(vector<int>& nums, int k) {
    int n = nums.size();
    deque<int> q;
    for (int i = 0; i < k; ++i) {
        while (!q.empty() && nums[i] >= nums[q.back()]) {
            q.pop_back();
        }
        q.push_back(i);
    }
    vector<int> ans = {nums[q.front()]};
    for (int i = k; i < n; ++i) {
        while (!q.empty() && nums[i] >= nums[q.back()]) {
            q.pop_back();
        }
        q.push_back(i);
        while (q.front() <= i - k) {
            q.pop_front();
        }
        ans.push_back(nums[q.front()]);
    }
    return ans;
```
