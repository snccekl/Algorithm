#  [第一题 每个字符最多出现两次的最长子字符串](https://leetcode.cn/problems/maximum-length-substring-with-two-occurrences/description/)
用滑动窗口，同时用一个哈希完成字母统计，这里注意原生的map不支持找最大值，故调用函数`max_element`，需要重载运算符
下为代码：
```cpp
class Solution {
public:
    static bool cmp_value(const pair<char, int> left,const pair<char,int> right){
	    return left.second < right.second;
    }
    int maximumLengthSubstring(string s) {
        unordered_map<char, int> count;
        int left = 0, right = 0, max_len = 0;
        while (right < s.size()) {
            count[s[right]]++;
            while (max_element(count.begin(), count.end(), cmp_value)->second > 2) {
                if (--count[s[left]] == 0) {
                    count.erase(s[left]);
                }
                left++;
            }
            max_len = std::max(max_len, right - left + 1);
            right++;
        }
        return max_len;
    }
};
```
# [第二题 执行操作使数据元素之和大于等于k](https://leetcode.cn/problems/apply-operations-to-make-sum-of-array-greater-than-or-equal-to-k/description/)
这里关键是一个贪心：一定是先加一到一个数后，再进行复制

之后就是枚举，这里做一个数学推导：

假设加到的数为m，再要其数据和大于等于k，需要的次数为$m - 1 + \lceil k / m \rceil - 1= m - 1 + \lfloor (k-1)/m \rfloor$

枚举即可
```cpp
class Solution {
public:
    int minOperations(int k) {
        int ans = k - 1;
        for(int i = 2;i<=k;i++){
            ans = min(i-1 + (k-1)/i, ans);
        }
        return ans;
    }
};
```
# [第三题 最高频率的ID](https://leetcode.cn/problems/most-frequent-ids/description/)
## 哈希+有序集合
- 用哈希表 $\textit{cnt}$ 记录$x=\textit{nums}[i]$ 的出现次数$\textit{cnt[x]}$（用$\textit{freq}$更新出现次数）
- 用有序集合记录$\textit{cnt[x]}$，以在`O(log n)`的时间下找到最大值同时作删除、插入
```cpp
class Solution {
public:
    vector<long long> mostFrequentIDs(vector<int>& nums, vector<int>& freq) {
        unordered_map<int, long long> cnt;
        multiset<long long> mset;
        vector<long long> ans;
        for(int i = 0;i<freq.size();i++){
            int x = nums[i];
            auto it = mset.find(cnt[x]);
            if(it != mset.end()){
                mset.erase(it);
            }
            cnt[x] += freq[i];
            mset.insert(cnt[x]);
            ans.push_back(*mset.rbegin());
        }
        return ans;
    }
};
```
## 哈希+懒删除堆
这一思路相比于上一思路，不需要有序集合，而只需要一个最大堆即可，如何维护数据的有效性：

最大堆的元素同时记录频率以及ID，当我作更新哈希表内数据时，先`push`该元素。之后对于堆顶，将其与哈希表内对应数据核对，若相同，则有效;否则无效，将其`pop`，再一直核对即可。
```cpp
class Solution {
public:
    vector<long long> mostFrequentIDs(vector<int>& nums, vector<int>& freq) {
        unordered_map<int, long long> cnt;
        priority_queue<pair<long long, int>> pq;
        vector<long long> ans;
        for(int i = 0;i<freq.size();i++){
            int x = nums[i];
            cnt[x] += freq[i];
            pq.emplace(cnt[x], x);
            while(pq.top().first != cnt[pq.top().second]){
                pq.pop();
            }
            ans.push_back(pq.top().first);
        }
        return ans;
    }
};
```
# [第四题 最长公共后缀查询](https://leetcode.cn/problems/longest-common-suffix-queries/description/)
## 字符串哈希
按灵神所说，这一方法不能保证一定对，不做阐述
## 字典树
这里直接搬灵神的了，自己也没什么其他要补的
倒着遍历 `s`，插入字典树。插入时，对于每个经过的节点，更新节点对应的最小字符串长度及其下标。
对于查询$s=\textit{wordsQuery}[i]$，仍然倒着遍历 `s`。在字典树上找到最后一个匹配的节点，那么该节点保存的下标就是答案。
```cpp
struct Node{
    Node* son[26]{};//有26个字母，就有26个分叉
    int min_l = INT_MAX;
    int i;
};
class Solution {
public:
    vector<int> stringIndices(vector<string>& wordsContainer, vector<string>& wordsQuery) {
        vector<int> ans;
        Node *root = new Node();
        for(int idx = 0;idx < wordsContainer.size();idx++){
            auto &s = wordsContainer[idx];
            int l = s.length();
            auto cur = root;
            if(l < cur->min_l){
                cur->min_l = l;
                cur->i = idx;
            }
            for(int i = l - 1;i>=0;i--){//倒序，为了方便匹配
                int b = s[i] - 'a';
                if(cur->son[b] == nullptr){
                    cur->son[b] = new Node();
                }
                cur = cur->son[b];
                if(l < cur->min_l){
                    cur->min_l = l;
                    cur->i = idx;
                }   
            }
        }

        for(auto &s: wordsQuery){
            auto cur = root;
            for(int i = s.length()-1;i>=0 && cur->son[s[i] - 'a'];i--){
                cur = cur->son[s[i] - 'a'];
            }
            ans.push_back(cur->i);
        }
        return ans;
    }
};
```
