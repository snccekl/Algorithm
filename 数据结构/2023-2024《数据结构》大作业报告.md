# 第一题
## 解题思路
本题使用并查集解决。
首先遍历一遍矩阵，当相邻两格均为岛屿时，认为二者连通，用并查集来维护二者的连通性。

枚举每一个网格由海变陆（当然，本身必须得是海），将其加入图中，判断其是否能使得其上下左右的连通块（或其中几个）相连。
为了简化并查集的操作，进行一个二维坐标向一维坐标的转化、

**注意类似于上方连通块与左方连通块本来就相连的情况**
时间复杂度：O(n^2)。即为遍历二维矩阵的时间复杂度。并查集find函数的时间可认为是O(1)的（进行过路径压缩）。
空间复杂度：O(n^2)
## 代码
**注意：代码中出现的max函数并非调库，而是自己实现**
并查集find函数代码（不做注释）
```cpp
int find(int x) {
    if (parent[x] != x){
        parent[x] = find(parent[x]);
    }
    return parent[x];
}
```

## 解题思路
使用Treap解决
每次插入新刷新的数据，然后通过Treap记录当前的最钟意评分，最后输出即可

时间复杂度：期望为O（nlogn），实际应观察数据
空间复杂度：为O(n),没有使用链表，而是使用多个一维数组
## 代码

}

# 第四题
## 解题思路
用两个最小堆，一个按照购买时间排序，另一个按照销量和购买时间（即题意）排序

时间复杂度：O(nlogn)
空间复杂度：O(n)（常数级）
