# 获得最小值
每一次固定右端点r，则只需要找前面最小的值即可，这可以在遍历中获得。
代码如下：
```cpp
int mn = 0;
int fix = 0;
for(int i = 1;i<=n;i++){
	fix = max(a[i] - mn, fix);
	mn = min(mn, a[i]);
}
```
# 贪心
这一思路的关键在于求最大子段和我只考虑为正的部分，如果出现了负数，直接从下一个数开始即可。
代码如下：
```cpp
int mx = 0;
int fix = 0;
for(int i = 1;i<=n;i++){
	mx = max(0, mx + a[i]);
	fix = max(fix, mx);
}
```
