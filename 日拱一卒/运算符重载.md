# operator
代码如下：
```cpp
struct book{
    int a,b;
    bool operator > (const book &u ) const{
        if(a == u.a){
            return b > u.b;
        }
        else{
            return a > u.a;
        }
    }
}p[200005];//重载大于
```
重载大于后，如果使用sort，需要修改排序规则，否则会出现无法排序的情况
代码如下：
```cpp
sort(p, p+n, greater<book>());
```
# cmp
作为函数使用，代码如下：
```cpp
bool cmp(int x,int y){
	return x % 10 > y % 10;
}
```
使用sort时，仍然需要改变排序规则
代码如下：
```cpp
sort(num,num+10,cmp);
```
