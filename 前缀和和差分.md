# 一维前缀和及差分
前缀和数组：$p_i = \sum_{j=1}^i a[i] = p_{i-1} + a[i]$ 
差分数组：$d[i] = a[i] - a[i-1]$   $\sum_{j=1}^i d[j] = a[i]$
差分可以完成$a[i]$的还原。
可以完成后缀区间的修改，因为只需要对某个$d[i]$做一次修改，就可以对后续所有$a[i]$的值完成修改。例如对$d[2]+x$，则$a[2]$以及之后的元素都会$+x$

# 二维前缀和及差分
前缀和数组：$p[i][j] = \sum_{t_i = 1}^i \sum_{t_j = 1}^j a[i][j] = p[i-1][j] + p[i][j-1] - p[i-1][j-1] + a[i][j]$
差分数组：$d[i][j] = a[i][j] - a[i-1][j] - a[i][j-1] + a[i-1][j-1]$
对应元素修改的方法：
思考一个矩阵即可。
```cpp
diff[x1][y1] += v;
diff[x1][y2+1] -= v;
diff[x2+1][y1] -= v;
diff[x2+1][y2+1] += v;
```
