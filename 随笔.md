# 输入输出
## 选择哪种输入输出函数
直接看示例：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main(){
	ios::sync_with_stdio(0), cin.tie(0), cout.tie(0);
	//1.scanf和printf
	//功能强大，但是不能取消同步流；
	//2.cin和cout
	//在取消同步流的情况下，输出存储在缓冲区，会比scanf和printf更快
}
```
## 换行
不要使用cout<<endl；
原因是：endl包含换行符+清空缓冲区（后者操作较慢）
使用cout<<'\n'
>遇到交互题的话endl会更好一点
## 输入
尽量在输入的时候只进行输入，而不进行其他操作
# 常数
尽量直接用const确定N的大小，而不用define（原因是define作为字符串替换可能替换后发生问题）
变量开在全局，会默认初始为0。关键是空间大，如果在main函数里，则用栈空间，小，易爆栈。
